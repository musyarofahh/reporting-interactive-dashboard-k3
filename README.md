# Interactive Dashboard for Integrated OHS Testing (K3)  

## 📌 Project Overview  
This project develops an **interactive dashboard** using **Looker Studio** to support integrated Occupational Health and Safety (K3) testing. The dashboard consolidates multiple data sources into a single platform, enabling clear visualization, monitoring, and analysis of workplace health and safety indicators.  

## 🎯 Objectives  
- Provide **real-time monitoring** of K3 testing results.  
- Identify and visualize **risk factors** that influence occupational health outcomes.  
- Support **decision-making** with transparent, data-driven insights.  
- Improve **reporting efficiency** and compliance tracking.  

## 🔑 Key Features  
- **Centralized Data Visualization**: Charts, tables, and KPIs for quick interpretation.  
- **Interactive Filters**: Drill down by department, location, or time period.  
- **Risk Factor Analysis**: Highlight correlations between workplace conditions and hazards.  
- **Performance Monitoring**: Track compliance levels, incident trends, and preventive actions.  
- **User-Friendly Interface**: Accessible design for both technical and non-technical stakeholders.  

## 🛠️ Tools & Technologies  
- **Looker Studio** for dashboard creation and visualization.  
- **Google Sheets / Excel** as primary data sources.  

## 📈 Expected Outcomes  
- Enhanced **transparency** in K3 implementation.  
- Improved **decision support** for management and safety officers.  
- Strengthened **preventive strategies** through data-driven insights.  
- A scalable framework for future **occupational health monitoring** projects.
